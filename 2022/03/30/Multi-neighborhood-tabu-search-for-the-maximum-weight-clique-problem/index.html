2 Summary

2.1 Introduction

古籍释义： 团（clique）是图论中的用语。对于给定图。其中，是图G的顶点集，E是图G的边集。图G的团就是一个两两之间有边的顶点集合。简单地说，团是G的一个完全子图。如果一个团不被其他任一团所包含，即它不是其他任一团的真子集，则称该团为图G的极大团（maximal clique）。顶点最多的极大团，称之为图G的最大团（maximum clique）。最大团问题的目标就是要找到给定图的最大团。（这就是MCP）

MCP是一个NP难问题。而MWCP是MCP问题的一个比较重要的generalization。MCP的最大团并不一定是MWCP的最大权团，并且最大权团问题的计算复杂度至少和最大团问题的一样。

有很多精确性算法可以解决一些小规模MWCP。对于大规模问题，可以用启发式算法来寻求近似解。另外还有一系列使用DIMACS图的MWCP基准实例被创造（这篇论文使用了这些实例）。

这篇论文提出的方法叫multi-neighborhood tabu search approach，中文我感觉是叫多邻域禁忌搜索算法？？为了有效地探索搜索空间，该算法结合了由三种移动引起的三个邻域。合并邻域的特殊性依赖于底层邻域的结合，而不是传统的基本邻域的顺序探索。在算法的每次迭代中，我们的标签搜索方法探索这三个邻域的联合，并选择总体上最佳允许的邻域解。该算法集成了一个专用的标签机制和一个随机重启策略。（这里看完论文以后回来再看一下！）

2.2 Model

首先初始化一个团，然后使用禁忌搜索算法来使其权重最大从而优化。下面是构造初始团的步骤：第一步，从图中随机选择一个seeding vertex ，并且将当前集团设置为由此单个顶点组成的集合；第二步，随机选一个不在团里面的顶点，并且顶点要和团中的所有顶点都相连接。第二步一直重复，直到找不到符合条件的顶点了。

局部搜索里面，邻域通常是通过一个move算子来定义的，这个算子将一个给定的解转换为一个相邻解，按照。

在MN/TS算法中，选用了一个一般的禁忌规则：如果一个顶点离开了目前的团，也就是通过SWAP或者DROP移动算子离开了当前解，下一个禁忌步长内就不允许这个顶点回到当前解中。而一个通过ADD或者SWAP移动算子加入到团中的顶点可以从其中自由被删除（无限制）。

因此，ADD移动算子不需要禁忌列表。可以直观地解释为，由于最大化团权重的目标，一个添加的顶点几乎没有机会被删除。

在禁忌搜索算法的迭代过程中，会出现侯选集中的全部对象都被禁忌，或有一对象被禁，但若解禁则其目标值将有非常大的下降情况。在这样的情况下，为了达到全局最优，我们会让一些禁忌对象重新可选。这种方法称为特赦，相应的规则称为特赦规则。（来自【算法】禁忌搜索算法(Tabu Search，TS)超详细通俗解析附C++代码实例 - 短短的路走走停停 - 博客园 (cnblogs.com)）

每次迭代，算法都会考核三个邻域，并且选择一个生成最有利的临近解的合适移动。

每次当前搜索被判定为陷入深度局部最优时，都会触发重启，这种情况被认为发生在超过最大允许的连续迭代次数而不提高群权重时。我们就叫这个数字为，搜索深度。

基本上，这个 multistart tabu search algorithm 根据以下两个步骤进行迭代，直到终止条件达到：

  • 生成一个新的初始点（see Section 2.2)
  • 使用禁忌搜索程序来改善，直到固定的深度达到了。

当算法达到预定的最大迭代次数()时就停止。
